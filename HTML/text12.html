Tasmota module

  • Send a Tasmota command from Berry
  • Create a Tasmota command in Berry
  • Timers

tasmota.cmd(“dimmer 60”)

def hello_from_berry() print(“Hello from Berry”)

end

tasmota.add_cmd(“hello”,hello_from_berry)

tasmota.set_timer(600,hello_from_berry)

tasmota.add_cron("*/10 * * * * *",hello_from_berry)


Notes:

Tasmota.cmd performs the same function as typing a command from the console.

Create a new tasmota command called ‘hello’ that can be used from the console or rule

Timers are the obvious mechanism to create future events. Berry execution can cease until the timer expires.

The cron function adds a second component to the standard cron time format.

This example calls the hello_from_berry function every 10 seconds.
