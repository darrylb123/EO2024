Program Execution

  • Berry programs in the file-system can be executed:
    • by the load() function from the Berry console
    • Run at startup if named ‘autoexec.be’ which can then load() other files.
    • From the Tasmota console using the ‘br’ command
    • From a tasmota rule

ON Time#Minute=0 DO br load(“midnight.be”) ENDON

ON DS18B20#Temperature<10

DO WEBSEND[192.168.10.3] br load(“TooCold.be”)


Notes:

The load() function looks for that filename or with a .be extension, loads and executes.

If the program has not been loaded previously, it is compiled to bytecode then executed. The bytecode is stored as the filename with a .bec extension.

This also applies if the file is newer than the bytecode file.

Berry VM can be restarted with the ‘brrestart’ console command
