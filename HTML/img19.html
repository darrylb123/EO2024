Notes:


Wave tank project for local university Phd student.

Variable speed drive runs the wave mechanism. Drive is soft started by a Berry program in one controller.

Relay modules control a pump for each wave tank to simulate tides with a berry function

ESP8266 Load cells weight the tanks to provide water level feedback.

Raspberry Pi Model A is Wifi AP, MQTT broker and overall web interface.
