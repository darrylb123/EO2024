Support Libraries

  • Berry has a number of modules that can be imported as required:
    • String
    • JSON
    • MATH
    • MQTT
    • Tasmota
    • GPIO

Notes:

String library contains the usual string functions, formatting, counting occurrences,search etc

JSON library contains function for serialising and deserialising JSON strings. Very useful as lots of calls to Tasmota returns a JSON string.

MATH

MQTT (publishing and subscribing)

Tasmota library contains many functions and classes for getting data and manipulating tasmota internals.
