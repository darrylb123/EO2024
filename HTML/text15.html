Driver with fast_loop

class my_driver

def every_100ms()

# called every 100ms via normal way

end

def fast_loop()

# called at each iteration, and needs to be registered separately and explicitly

end

def init()

# register fast_loop method

tasmota.add_fast_loop(/-> self.fast_loop())

end

end

tasmota.add_driver(my_driver()) # register driver


Notes:

Example from the documentation showing the overridden every_100ms() method

The Fast_loop() is added in the class init() method

The / arrow is the creation of a closure, in Berry a closure is best explained as a synthetic function pointer.
