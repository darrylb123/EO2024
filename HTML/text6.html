Commands and Rules

  • Command
  • Rule
  • There are other triggers depending on the peripherals configured

POWER1 ON

websend [192.23.1.20] power1 0

ON POWER1#STATE=1 DO RULETIMER1 600 ENDON

ON RULES#TIMER=1 DO POWER1 0 ENDON

ON ANALOG#A0div10 DO DIMMER %value% ENDON


Notes:

In the command example. Assuming a relay has been configured. POWER1 ON turns on the first relay

Using websend allows peer to peer commands

An example rule

When POWER1 is turned on a timer is loaded with 600 (seconds)

When the timer expires the relay is turned off

When ANALOG A0 changes more than 1% adjust the dimmer by the returned value

These examples do not need anything else to operate.
